Quickly bootstrap projects with a consistent `.github` setup including custom AI agents, instructions, and tools.

## Usage

Install globally:

```bash
cargo install --git https://github.com/Amund-Fremming/wraptor
```

Run:

```bash
wraptor
```

Select an option:

1. **Wrap existing project** - Copy a project into `wrapped-<name>` with `.github` config
2. **Create empty wrapper** - Create new `wrapped-<name>` folder with `.github` config

Press `Ctrl+C` to exit.
